# waybar

My Waybar setup

https://github.com/Alexays/Waybar

Copy the contents of the repository folder to `~/.config/waybar/`. The following files can be ignored:

- `README.md`
- `.gitignore`
- `.git`

## Installation (Omarchy/Arch Linux)

_Omarchy comes pre-installed with Waybar, so no istallation is needed_

```sh
pacman -S \
  gtkmm3 \
  jsoncpp \
  libsigc++ \
  fmt \
  wayland \
  chrono-date \
  spdlog \
  gtk3 \
  gobject-introspection \
  libgirepository \
  libpulse \
  libnl \
  libappindicator-gtk3 \
  libdbusmenu-gtk3 \
  libmpdclient \
  sndio \
  libevdev \
  libxkbcommon \
  upower \
  meson \
  cmake \
  scdoc \
  wayland-protocols \
  glib2-devel
```
